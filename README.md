# Pre-requisites for the projects.
    # **Below are the required pre-requisites**

    1. Java 17.
    2. Maven 3.9.
    3. Cassandra 4.1.3


# Steps to run the project locally

    1. mvn clean compile
    2. Configure your cassandra details in application.yml
    3. Run Investment Application class.
